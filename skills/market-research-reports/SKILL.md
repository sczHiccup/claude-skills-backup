# 市场研究报告

## 概述

市场研究报告是全面的战略文件，分析行业、市场和竞争格局，以为商业决策、投资策略和战略规划提供信息。此技能生成 **专业级别的 50+ 页报告**，具有丰富的视觉内容，模仿顶级咨询公司如麦肯锡、BCG、贝恩、Gartner 和 Forrester 的交付成果。

**主要特点：**
- **全面的长度**：报告设计为 50+ 页，没有令牌限制
- **丰富的视觉内容**：开始时生成 5-6 个关键图表（在写作过程中根据需要添加更多）
- **数据驱动的分析**：与 research-lookup 深度集成以获取市场数据
- **多框架方法**：波特五力、PESTLE、SWOT、BCG 矩阵、TAM/SAM/SOM
- **专业格式**：咨询公司质量的排版、颜色和布局
- **可操作的建议**：战略重点与实施路线图

**输出格式：** LaTeX，具有专业样式，编译为 PDF。使用 `market_research.sty` 样式包以确保一致的专业格式。

## 何时使用此技能

当以下情况时应使用此技能：
- 为投资决策创建全面的市场分析
- 为战略规划开发行业报告
- 分析竞争格局和市场动态
- 进行市场规模评估（TAM/SAM/SOM）
- 评估市场进入机会
- 为并购活动准备尽职调查材料
- 创建行业定位的思想领导内容
- 开发市场进入战略文档
- 分析监管和政策对市场的影响
- 为新产品发布建立商业案例

## 工作流程

### 第一阶段：研究与数据收集

**步骤 1：定义范围**
- 澄清市场定义
- 设置地理边界
- 确定时间范围
- 确定需要回答的关键问题

**步骤 2：进行深入研究**

广泛使用 `research-lookup` 收集市场数据：

```bash
# 市场规模和增长数据
python skills/research-lookup/scripts/research_lookup.py \
  "当前 [MARKET] 行业的市场规模和预测增长率是多少？包括 TAM、SAM、SOM 估算和 CAGR 预测"

# 竞争格局
python skills/research-lookup/scripts/research_lookup.py \
  "在 [MARKET] 市场中，前 10 名竞争者是谁？他们的市场份额和竞争定位是什么？"

# 行业趋势
python skills/research-lookup/scripts/research_lookup.py \
  "2024-2030 年 [MARKET] 行业的主要趋势和增长驱动因素是什么？"

# 监管环境
python skills/research-lookup/scripts/research_lookup.py \
  "影响 [MARKET] 行业的关键法规和政策变化是什么？"
```

**步骤 3：数据组织**
- 创建 `sources/` 文件夹，存放研究笔记
- 按部分组织数据
- 确定数据缺口
- 根据需要进行后续研究

### 第二阶段：分析与框架应用

**步骤 4：应用分析框架**

对每个框架进行结构化分析：

- **市场规模**：TAM → SAM → SOM，清晰假设
- **波特五力**：对每个力量进行高/中/低评级并给出理由
- **PESTLE**：分析每个维度的趋势和影响
- **SWOT**：内部优势/劣势，外部机会/威胁
- **竞争定位**：定义轴线，绘制竞争者

**步骤 5：发展见解**
- 将发现综合为关键见解
- 确定战略影响
- 制定建议
- 优先考虑机会

## 质量标准

### 页数目标

| 部分 | 最低页数 | 目标页数 |
|------|----------|----------|
| 前言 | 4 | 5 |
| 市场概述 | 4 | 5 |

| 行业驱动因素 | 4 | 6 |
| 竞争格局 | 5 | 7 |
| 客户分析 | 3 | 5 |
| 技术格局 | 3 | 5 |
| 监管环境 | 2 | 4 |
| 风险分析 | 2 | 4 |
| 战略建议 | 3 | 5 |
| 实施路线图 | 2 | 4 |
| 投资论点 | 2 | 4 |
| 后记 | 4 | 5 |
| **总计** | **43** | **66** |

### 视觉质量要求

- **分辨率**：所有图像至少 300 DPI
- **格式**：光栅图使用 PNG，矢量图使用 PDF
- **可访问性**：色盲友好的调色板
- **一致性**：整个报告使用相同的配色方案
- **标签**：所有轴、图例和数据点都要标注
- **来源归属**：图形标题中引用来源

### 数据质量要求

- **时效性**：数据不超过 2 年（优先考虑当前年份）
- **来源**：所有统计数据都要归属到具体来源
- **验证**：尽可能交叉参考多个来源
- **假设**：所有预测都要说明基础假设
- **限制**：承认数据的限制和缺口

### 写作质量要求

- **客观性**：呈现平衡的分析，承认不确定性
- **清晰性**：避免行话，定义技术术语
- **准确性**：使用具体数字而非模糊定性
- **结构**：清晰的标题，逻辑流畅，过渡自然
- **可操作性**：建议具体且可实施

## LaTeX 格式

### 使用样式包

`market_research.sty` 包提供专业格式。在文档中包含它：

```latex
\documentclass[11pt,letterpaper]{report}
\usepackage{market_research}
```

### 框环境

使用彩色框突出显示关键内容：

```latex
% 关键见解框（蓝色）
\begin{keyinsightbox}[关键发现]
市场预计将在 2030 年之前以 15.3% 的 CAGR 增长。
\end{keyinsightbox}

% 市场数据框（绿色）
\begin{marketdatabox}[市场快照]
\begin{itemize}
    \item 市场规模（2024）：\$45.2B
    \item 预计规模（2030）：\$98.7B
    \item CAGR：15.3%
\end{itemize}
\end{marketdatabox}

% 风险框（橙色/警告）
\begin{riskbox}[关键风险]
监管变化可能影响 40% 的市场参与者。
\end{riskbox}

% 建议框（紫色）
\begin{recommendationbox}[战略建议]
优先考虑进入亚太地区市场。
\end{recommendationbox}

% 提示框（灰色）
\begin{calloutbox}[定义]
TAM（总可寻址市场）代表总收入机会。
\end{calloutbox}
```

### 图形格式

```latex
\begin{figure}[htbp]
\centering
\includegraphics[width=0.9\textwidth]{../figures/market_growth.png}
\caption{市场增长轨迹（2020-2030）。来源：行业分析，公司数据。}
\label{fig:market_growth}
\end{figure}
```

### 表格格式


## 故障排除

### 常见问题

**问题**：报告页数不足 50 页
- **解决方案**：扩展附录中的数据表，添加更多详细的公司简介，包含额外的区域分解

**问题**：视觉未渲染
- **解决方案**：检查 LaTeX 中的文件路径，确保图像在 figures/ 文件夹中，验证文件扩展名

**问题**：书目缺少条目
- **解决方案**：在第一次 xelatex 通过后运行 bibtex，检查 .bib 文件中的语法错误

**问题**：表格/图形溢出
- **解决方案**：使用 `\resizebox` 或 `adjustbox` 包，减少图像宽度百分比

**问题**：生成的视觉质量差
- **解决方案**：使用 `--doc-type report` 标志，增加迭代次数 `--iterations 5`
